ZPT-JS reference - Evaluation order

The order evaluation of attributes in ZPT-JS is not equal than ZPT's. The new order is:

  1. data-repeat
  2. data-on-error
  3. data-define-macro
  4. data-define
  5. data-condition
  6. data-omit-tag
  7. data-replace
  8. data-attributes
  9. data-content
  10. content
  11. data-use-macro
