Simple format

  1. format: lowerCase 'Test' should be a lowercase
  2. format: upperCase 'Test' should be an uppercase
  3. format: lowerCase aString should be a lowercase
  4. format: upperCase aString should be an uppercase

Format with arguments

  1. format: fix 1.371 2 should be a number fixed to 2
  2. format: fix number fixTo should be a number fixed to 1
  3. format: fix (/: 1 3) 2 should be a number fixed to 2

Custom formatters

  1. format: customFormatter 150 should be money!
  2. format: customFormatter number should be money!
  3. format: customFormatterName number should be money!
